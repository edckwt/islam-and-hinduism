== Description ==

Islam-hinduism.com intends to be one of the main online sources of information about Hindu-Muslim interaction.

Features:

* Getting the content (articles, videos, audios) of Islam-hinduism.com.
* Using Json API
* Using Cron Job

PS: All the websites owned by EDC [(E-Da`wah Committee)](http://www.edc.org.kw), For Support click to [Islam.com.kw](http://www.islam.com.kw/DawahApps/).

== Installation ==

* Upload the Plugin to your blog, Activate it.
* Select the language.
* Choose the site.
* Then the content will be stored automatically in your database under the category you choose or according to the original category.
* The plugin will work automatically using Cron Job to get the recent updates daily or according to the time you set.
